About Parkinson’s Disease

A short, readable primer for judges and users — concise, accurate, and educational. This page is for information only.

What is Parkinson’s disease?

Parkinson’s disease (PD) is a progressive neurodegenerative condition that mainly affects movement and motor control. Symptoms include tremor, bradykinesia (slowness of movement), rigidity, and postural instability. Non-motor signs can include sleep disturbances, smell loss, and changes in speech and voice.

Why voice?

Alterations in speech and voice are among early, measurable signs in many people with PD. Acoustic markers like fundamental frequency, jitter (pitch microvariability), shimmer (amplitude microvariability), and harmonic-to-noise ratio can change before obvious motor symptoms.

What this demo does

  • Uses a small public dataset (Kaggle) of recorded voice-derived features.
  • Shows a simple machine learning model (Decision Tree) trained for educational demonstration.
  • Returns an educational prediction and a brief feature-wise explanation not a medical diagnosis.

Limitations & ethics

This web demo is not a medical device. It uses a limited dataset and a simple model. It cannot replace clinical evaluation by neurologists. Inputs are numeric features, not raw audio, and no personal data is stored by default.

Key voice features we use

  • Fo / Fhi / Flo — fundamental frequency (pitch averages & range)
  • Jitter — pitch instability (cycle-to-cycle)
  • Shimmer — amplitude instability
  • HNR — harmonic-to-noise (clarity)
  • DFA — complexity / long-range correlation

Quick QA for judges

How does the demo work?

Frontend posts 7 numeric features to a Flask API (/predict). Backend scales the numbers and uses a saved model to predict. The UI shows prediction + a feature-level note.

Is this accurate?

Not for clinical use. Small dataset and a simple model suitable for learning and prototype demonstrations only.

Privacy

No audio or personal data stored by default. Inputs are numeric features only.
