What is Parkinson’s disease?

Parkinson’s disease is a progressive neurodegenerative disorder affecting movement and speech. Early changes in voice stability and clarity can be quantified through acoustic features such as fundamental frequency, jitter, shimmer, and noise ratios.

In this demo we explore how such voice features from a public dataset can be analyzed with machine learning to produce an educational prediction.

How the model works

Public Kaggle dataset → feature engineering → model training → Flask API → browser UI

Dataset

Biomarkers voice dataset from Kaggle with 20+ acoustic features and labels (status).

Features

Pitch (Fo, Fhi, Flo), jitter, shimmer, noise ratios (NHR/HNR), DFA, etc.

ML Model

Trained with scikit-learn. Saved as model.pkl and served via Flask.

API

POST /predict endpoint returns JSON with predicted class.

Frontend

Interactive form with tooltips, results panel, and neat visuals.

Important

Educational use only

This demo is not a medical device. It uses a small public dataset and a simple ML model to illustrate concepts.

Why voice features?

Early PD can alter phonation stability. Acoustic features like jitter, shimmer, and HNR capture subtle dysphonia patterns.

Privacy

No personal data is stored. Inputs are numeric feature values, not audio recordings.
